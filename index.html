HTML Table Tags Overview
Tag/Selector/Property Name Description
<table> Table Defines a table, organizing data in rows and columns.
<tr> Table Row Represents a row inside the table, containing cells.
<th> Table Header Cell Defines a cell as a header in the table; usually bold and centered.
<td> Table Data Cell Represents a standard data cell in the table.
<caption> Table Caption Provides a title or description for the whole table.
<thead> Table Head Groups the header content in the table.
<tbody> Table Body Groups the body content in the table, separating it from the header or footer.
<tfoot> Table Footer Groups footer content in the table, often used for totals.
<col> Table Column Specifies column properties that can be applied to multiple columns.
<colgroup> Column Group Groups together a set of columns, allowing collective formatting or properties.
Other HTML Tags
Tag Name/Selector Description
<ul> Unordered List Creates a bulleted list.
<ol> Ordered List Creates a numbered list.
<li> List Item Represents a list item in ordered or unordered lists.
<dl> Description List Defines a list of terms and descriptions, also called definition lists.
<dt> Definition Term Specifies a term in a description list.
<dd> Description Details Provides the description for a term in a description list.
<blockquote> Block Quote Indicates quoted text from another source.
<code> Code Represents a fragment of computer code.
<em> Emphasized Text Makes text emphasized, typically italic.
<mark> Marked/Highlighted Text Highlights part of the text in a paragraph (highlight effect).
<main> Main Content Defines the main content area unique to the document.
CSS Selectors and Specificity
Tag/Selector/Property Name Description
* Universal Selector Selects all elements in the HTML document.
element Type Selector Selects all elements of a specific type, e.g., p for all paragraphs.
.class Class Selector Selects elements with a specific class attribute.
#id ID Selector Selects a single element with a unique ID.
[attr] / [attr=value] Attribute Selector Selects elements based on attribute presence/value (e.g., [type="text"]).
Common CSS Properties
Property Name Description
color Text Color Sets the color of the text.
background Background Sets background color, image, position, etc.
margin Margin Controls the space outside elements.
padding Padding Controls the space inside elements, around content.
border Border Sets an element's border style, width, color, etc.
font-size Font Size Sets text size.
font-family Font Family Specifies typeface(s) for text.
font-weight Font Weight Sets boldness of the text.
text-align Text Alignment Aligns text: left, right, center, or justify.
line-height Line Height Sets vertical spacing between lines of text.
display Display Controls layout rendering (block, inline, flex, etc).
position Position Specifies positioning method (static, relative, etc).
width/height Width/Height Sets width and/or height of elements.
overflow Overflow Handles content overflow (visible, hidden, scroll).
list-style List Style Sets bullet/number type, marker image, position.
